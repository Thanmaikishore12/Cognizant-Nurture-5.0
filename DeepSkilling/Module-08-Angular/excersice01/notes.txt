1. angular.json
Main Angular configuration file. It stores project build, serve, test and asset settings.

2. tsconfig.json
Main TypeScript configuration file used throughout the project.

3. tsconfig.app.json
TypeScript configuration specifically for compiling the Angular application.

4. package.json
Contains project information, dependencies, scripts and package versions.

5. src/main.ts
Entry point of the Angular application. It bootstraps the application.

6. src/app/app.config.ts
Contains the application configuration such as routing and providers.

7. src/app/app.ts
Root component of the Angular application.

8. src/index.html
Main HTML page where the Angular application is loaded.
